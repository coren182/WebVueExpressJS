Cách chạy Source Code:

Mở 2 tab terminal dẫn đến trong thư mực source : 
- Một tab chạy lệnh npm run serve
- Một tab kia chạy lệnh npm run dev

Nếu máy thầy sử dụng chạy MacOS khi chạy lại 2 lệnh trên lần 2 có thể sẽ bị trùng port nên khi bao lỗi thầy chạy lệnh killall node -9.
